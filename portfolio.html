Project 1:The Film Vault - Movie Database and Viewer

Scope & Objectives:

The project aims to create a web application called "The Film Vault" that serves as a movie database and viewer. Users can search for movies, view details about movies currently playing, log movies they have watched, and add new movies to their watched list. The main objectives include providing a user-friendly interface to explore movies, interact with a movie database, and manage their watched movie list.

Screen shots

home page

search feature

logging movies

Technologies used:

  • HTML5: Markup language for structuring the web page
  • Styling language for designing the visual appearance
  • JavaScript: Programming language for interactivity and dynamic behavior
  • The Movie Database (TMDB) API: Used to fetch movie data and information
  • Fetch API: JavaScript's built-in API for making network requests

    • Features

    1. Movie Database Viewer: Displays information about movies currently playing, including title, poster, IMDb rating, and a link to IMDb for more details.
    2. Movie Search: Allows users to search for movies by entering a query in the search bar. Displays search results with movie details.
    3. Movies Watched: Users can log movies they have watched, providing the title and year. Movies are displayed in a table format.
    4. Add Movie: Enables users to add new movies to their watched list by entering the movie title and year.
    5. Responsive Design: The application is designed to work well on various screen sizes, from desktop to mobile.



    Project 2:Airport Airplane Tracker

    Scope & Objectives:

    The project's scope encompasses the development of an application named "Airport Airplane Tracker" designed to offer users real-time flight information and a user-friendly experience. The core objectives of this project are to provide users with immediate access to essential flight details, including flight numbers, airport locations, and statuses. Additionally, the application will greet users warmly based on the current time and display a real-time clock. The inclusion of an aesthetically pleasing background image enhances the visual appeal of the application. Finally, the project aims to facilitate the management of flight data, allowing for the addition of new flight information to ensure the application's relevance and usability.

    The "Airport Airplane Tracker" application targets travelers, airport personnel, and aviation enthusiasts, aiming to deliver timely and accurate flight information while ensuring an enjoyable and efficient user experience.

    Technologies used:

  • Java: To create the GUI,interactivity and display appropriate data and information

    • Features

    1. Real-time Time Display: Shows the current time.
    2. Greeting Display: Greets the user based on the current time of day (morning, afternoon, evening).
    3. Flight Status Checker: Allows users to enter a flight number and check its status (airport and status).
    4. Flight Status Display: Displays flight information (flight number, airport, and status) after checking.
    5. Background Image: Provides a background image for the application.
